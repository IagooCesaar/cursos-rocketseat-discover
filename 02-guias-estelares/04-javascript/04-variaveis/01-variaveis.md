# Variáveis

- São nomes simbólicos para "receber" determinado valor!
- Também são utilizadas como "atalhos" para código (funções)
- Identificadores (identifiers)

> Podem ser utilizadas três palavras chaves para se criar uma variável:
>
> - var
> - let
> - const

Exemplo com `var`:

```js
var clima = "quente";
console.log(clima);

clima = "frio";
console.log(clima);
```

Exemplo com `let`:

```js
let clima = "quente";
console.log(clima);

clima = "frio";
console.log(clima);
```

Exemplo com `const`:

```js
const clima = "quente";
console.log(clima);

clima = "frio"; //resultará em erro pois constantes não devem ser alteradas
console.log(clima);
```
